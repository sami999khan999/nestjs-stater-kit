## üéØ Common Use Cases

### Use Case 1: Track User Signup

**Documents**:

- [ANALYTICS_QUICK_REFERENCE.md](./ANALYTICS_QUICK_REFERENCE.md) - "Track Signup"
- [ANALYTICS_AUTH_INTEGRATION_GUIDE.md](./ANALYTICS_AUTH_INTEGRATION_GUIDE.md) - Step 3

**Code**:

```typescript
await this.analytics.trackSignup(user.id, user.email, {
  provider: 'email',
  source: 'web',
});
```

---

### Use Case 2: Track Purchase

**Documents**:

- [ANALYTICS_QUICK_REFERENCE.md](./ANALYTICS_QUICK_REFERENCE.md) - "Track Conversion"
- [src/analytics/examples/ecommerce-integration.example.ts](./src/analytics/examples/ecommerce-integration.example.ts)

**Code**:

```typescript
await this.analytics.trackConversion({
  conversionName: 'purchase',
  value: 99.99,
  currency: 'USD',
  userId: user.id,
  userEmail: user.email,
});
```

---

### Use Case 3: Track Custom Event

**Documents**:

- [ANALYTICS_QUICK_REFERENCE.md](./ANALYTICS_QUICK_REFERENCE.md) - "Track Custom Event"

**Code**:

```typescript
await this.analytics.trackEvent({
  eventName: 'button_clicked',
  userId: user.id,
  properties: { buttonName: 'subscribe' },
});
```

---

### Use Case 4: Use Decorator

**Documents**:

- [ANALYTICS_QUICK_REFERENCE.md](./ANALYTICS_QUICK_REFERENCE.md) - "Using Decorators"

**Code**:

```typescript
@Post()
@TrackEvent('post_created', { category: 'content' })
async createPost(@Body() dto: CreatePostDto) {
  // Event tracked automatically
}
```
